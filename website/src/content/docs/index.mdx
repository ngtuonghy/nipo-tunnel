## Why Nipo?

<CardGrid stagger>
	<Card title="Interactive TUI" icon="laptop">
		Displays real-time bandwidth, connection status, and logs right in your terminal.
	</Card>
	<Card title="Multi-Tunneling" icon="rocket">
		Expose multiple ports simultaneously via a single `nipo.yml` configuration file.
	</Card>
	<Card title="Automatic Setup" icon="setting">
		Instantly downloads the native `cloudflared` daemon on the first run. No account required.
	</Card>
	<Card title="Fast & Free" icon="star">
		Powered by Cloudflare infrastructure to provide secure, high-speed HTTP tunnels globally.
	</Card>
</CardGrid>
